# Sales Workflow — IMI

## Fluxo premium
Lead → Qualificação → Entendimento → Curadoria → Apresentação → Visita → Negociação → Due Diligence → Fechamento → Pós-venda

## Qualificação
- Perguntar: objetivo, timeline, orçamento, estrutura familiar
- Classificar: Investidor / Comprador / Incorporador / Corretor
- Score: Quente (>80) / Morno (50-80) / Frio (<50)

## Objeções
| Objeção | Framework |
|---------|-----------|
| "Está caro" | Reancoragem: custo de não fazer |
| "Preciso pensar" | Especificar: "o que avaliar?" |
| "Tenho outro corretor" | Diferenciação: consultoria integrada |
| "Mercado ruim" | Dados: análise de ciclo + oportunidade |
| "Faço sozinho" | Risco: compliance, tributário, jurídico |

## Follow-up cadence
- D+1: Agradecimento + resumo (WhatsApp)
- D+3: Conteúdo relevante
- D+7: Check-in + nova informação
- D+14: Reengajamento com caso/oportunidade
- D+30: Nurturing (newsletter, convite evento)
