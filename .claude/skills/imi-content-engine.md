# Content Engine — IMI

## Workflow
1. Receber BRIEF (tema + objetivo + persona alvo)
2. Gerar ARTIGO BASE (1200-2000 palavras, tom analítico)
3. Derivar para cada canal

## Derivações
### LinkedIn: analítico, thought leadership
- Post curto (150-300 palavras) com hook viral
- Artigo longo (800-1500 palavras)
- Carrossel (8-12 slides)

### Instagram: visual premium, educativo
- Carrossel educativo (8-10 slides)
- Caption com storytelling (2000 chars max)
- Reels script (30-60s)

### Blog/Site: institucional, SEO
- Artigo com H2/H3, schema JSON-LD
- Meta title + description otimizados

### Newsletter: exclusivo, consultivo
- Assunto provocativo, corpo 400-600 palavras, 1 insight, CTA

### WhatsApp: profissional mas acessível
- Mensagem curta (500 chars max), link para conteúdo completo

## Hierarquia: Educativo → Institucional → Cases → Produto → Mercado
