# Write Tests Skill

## Objetivo

Criar testes:
- legíveis
- previsíveis
- rápidos
- isolados
- sustentáveis

---

## Evitar

- mocks desnecessários
- testes frágeis
- duplicação
- dependência entre testes

---

## Priorizar

- testes de regra de negócio
- cenários críticos
- edge cases
- regressões
