# Code Review Skill

## Revisar

- bugs
- edge cases
- segurança
- performance
- tipagem
- arquitetura
- duplicação
- dead code
- complexidade desnecessária

---

## Verificar

- mudanças fora do escopo
- quebra de contrato
- imports incorretos
- acoplamento excessivo
- código inseguro
- regressões

---

## Resultado

1. Bloqueadores
2. Melhorias recomendadas
3. Aprovação ou reprovação
