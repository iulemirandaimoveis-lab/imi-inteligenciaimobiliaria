# Architecture Review Skill

## Revisar

- acoplamento
- separação de responsabilidades
- modularidade
- escalabilidade
- dependências cíclicas
- impacto sistêmico
- manutenção
- compatibilidade

---

## Evitar

- arquitetura desnecessária
- abstrações prematuras
- padrões excessivos
- complexidade artificial
