# Debug Production Skill

## Processo

1. minimizar impacto
2. identificar sintomas
3. identificar causa raiz
4. validar escopo
5. corrigir com mínimo risco
6. validar regressões

---

## Nunca

- expor dados sensíveis
- alterar produção sem necessidade
- executar mudanças destrutivas
- assumir causa sem evidência
